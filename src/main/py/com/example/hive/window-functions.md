### 1. What are the best-selling and the second best-selling products in every category?
```roomsql
SELECT
    product,
    category,
    revenue
FROM (
    SELECT
    product,
    category,
    revenue,
    dense_rank() OVER (PARTITION BY category ORDER BY revenue DESC) as rank
    FROM PRODUCT_REVENUE
) tmp
WHERE
rank <= 2
```
